🐙 Git Cheatsheet

Most common Git commands grouped by workflow

SETUP

  • git config --global user.name "Your Name" – Set name for commits
  • git config --global user.email "you@example.com" – Set email for commits
  • git config --global color.ui auto – Enable colorized output

SETUP & INIT

  • git init – Create a new Git repository
  • git clone [url] – Clone an existing repo

STAGE & SNAPSHOT

  • git status – Check repo status
  • git add [file] – Stage file
  • git add . – Stage all changes
  • git commit -m "message" – Commit staged changes
  • git reset [file] – Unstage a file
  • git diff – Show unstaged changes

BRANCH & MERGE

  • git branch – List branches
  • git branch [name] – Create new branch
  • git checkout [name] – Switch branches
  • git checkout -b [name] – Create and switch
  • git merge [branch] – Merge into current branch
  • git branch -d [name] – Delete a branch

INSPECT & COMPARE

  • git log – View commit history
  • git log --oneline – Condensed log
  • git diff – Show differences
  • git show [commit] – Show details of a commit

SHARE & UPDATE

  • git remote -v – List remotes
  • git fetch – Get latest from remote
  • git pull – Fetch and merge
  • git push – Push local to remote
  • git push -u origin [branch] – Push new branch

REWRITE HISTORY

  • git commit --amend – Modify last commit
  • git rebase [branch] – Reapply commits on another base
  • git reset --hard [commit] – Reset to specific commit

TEMPORARY COMMITS

  • git stash – Save changes temporarily
  • git stash pop – Reapply stash
  • git stash list – View stashes
