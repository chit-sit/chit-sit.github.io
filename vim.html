Vim Cheatsheet

Command Description
i Enter insert mode
Esc Exit insert mode
:w Save the file
:q Quit Vim
:wq Save and quit
dd Delete the current line
yy Copy (yank) the current line
p Paste after the cursor
u Undo last change
/text Search for "text"
:help Open Vim help
