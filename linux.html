CommandDescription
ls -lList files in long format
cd /pathChange directory
pwdShow current directory
mkdir myfolderCreate a directory
rm filenameDelete a file
rm -r folderDelete a folder recursively
cp file1 file2Copy file1 to file2
mv file1 newnameRename or move file
cat filePrint file contents
topMonitor running processes
chmod +x script.shMake a file executable
sudo commandRun command as superuser
sudo lsof -i :command to check what process running on specific port number, example command for port 3000 = sudo lsof -i :3000
sudo df -hcommand to check total disk space for mounted filesystems
ping command to test network connectivity between your machine and another host (like a server, website, or device), Example usage : ping google.com or ping 192.168.100.1
traceroute command to trace the network path that packets take from your computer to the target host, helpful to identify all the hops (routers/servers), network latency, where delays or failures occur in the route , Example usage : traceroute google.com
touch filename.txtcommand to create empty files
sudo fdisk -lcommand to show list of partition
uname -acommand to displays detailed system information about your machine, including the kernel version, hostname, hardware architecture, and operating system details.
dateTo print the current date and time in Linux
